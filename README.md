# Sarcasm Text Classification
Proyek ini bertujuan untuk mengidentifikasi apakah sebuah teks bersifat sarkastik atau tidak. Dengan menggunakan teknik pemrosesan bahasa alami (NLP), proyek ini memanfaatkan model pembelajaran mesin untuk mengklasifikasikan teks dengan akurasi tinggi.
